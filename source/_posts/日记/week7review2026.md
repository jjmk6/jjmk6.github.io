# 第7周复盘（2月10日 ~ 2月16日）

## 本周概览

本周是从学校回家的一周，完成了井盖论文初稿、红学爬虫跑通，但整体处于"多而散"的状态——买了大量 token 额度却找不到抓手，项目进展零散。回家后作息混乱，运动完全中断，处于一种"想做的事太多但无从下手"的焦虑状态。

## 关键成果

1. **井盖论文** - 完成初稿，写成 LaTeX 形式（但科研绘图、实验模块图待完善）
2. **红学爬虫** - 跑通基础功能，后续升级版爬取约 1000 个视频和 10 万条评论
3. **工具升级** - Claude Code 升级到 Opus 4.6，安装 mem 插件，配置智谱大龙虾
4. **知识库可视化** - 用 Kimi 制作了项目 dashboard（论文数据采集、竞赛分配、磁盘归档）
5. **GitHub 仓库** - 为文献阅读助手开仓，整合 80+ 篇文献

## 模式识别

### 精力/状态波动
- 平均精力：**6.0/10**
- 平均情绪：**6.0/10**
- 最高精力：无显著波动，整体处于及格线
- 最低精力：感冒期间（2月10日），身体不适

### 做得顺的事
- **工具折腾** - 配置各种 AI 工具（Claude Code、Kimi Code、GLM、智谱大龙虾）
- **可视化** - Kimi 的 UI 生成能力很强，快速做出 dashboard
- **爬虫突破** - 红学爬虫终于跑通，虽然过程中反复修改

### 阻力大的事
- **早睡早起** - 每天都说要从明天开始，但连续失败
- **运动** - 整周只有一天搬砖算运动，散步习惯完全中断
- **深度工作** - 面对多个项目（井盖论文、违建论文、祖师爷论文、爬虫）陷入选择困难

### 反复出现的卡点
1. **熵增不熵减** - 不断开新项目、买新 token，但缺乏复盘总结，知识沉淀不足
2. **抓手缺失** - 2月13日对着五六条命令行发呆，买了很多 token 但不知道让它们干什么
3. **多而散** - 同时推进 5 个方向（红学爬虫、井盖论文、祖师爷论文、违建论文、技能研究），每个都浅尝辄止

## 本周感悟

> **大模型时代的核心用户（20%）的特征不是"用得多"，而是把 token 转化成了可交付的成果。**

> **我们从一个思考者变成决策者，能做的事情越来越多了，也越来越忙了，消耗着无数的 token 为自己的思想建设。**

> **每天都在让很多大模型帮我去跑一些项目，不断在做熵增的过程，然后得到一些结论之后就不怎么管之前做过的一些事情了，没有进行总结归纳整理这种熵减的算法。**

## 下周调整

1. **熵减优先** - 每天必须做复盘总结，把项目产出整理成可交付文档（而不是跑完就扔）
2. **单一抓手** - 每天只选 1 个核心项目深度推进，其他项目暂停或限流
3. **作息恢复** - 23:00 前关机，7:30 起床，用闹钟强制约束
4. **运动重启** - 每天晚饭后散步 30 分钟，不带手机
5. **工具收敛** - 停止购买新 token/API，用完手头额度再评估
